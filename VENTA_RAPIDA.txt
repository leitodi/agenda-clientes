================================================================================
                    RESUMEN: CÓMO VENDER ESTE PROYECTO
================================================================================

PASO 1: Prepara el proyecto
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Abre la carpeta "Agenda de turnos" y ejecuta:
  → Haz doble clic en "PREPARAR_PARA_VENTA.bat"

Esto limpiará archivos innecesarios y dejará el proyecto listo.


PASO 2: Comprime la carpeta
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Opción A - Por Email (Windows):
  1. Haz clic derecho en "Agenda de turnos"
  2. "Enviar a" → "Carpeta comprimida"
  3. Se crea un archivo .ZIP
  4. Envía ese archivo por email

Opción B - Por USB:
  1. Copia la carpeta a una USB
  2. Entrega la USB al cliente

Opción C - Por Google Drive:
  1. Sube la carpeta a Google Drive
  2. Comparte el enlace con el cliente


PASO 3: Instrucciones para el cliente
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

El cliente recibe la carpeta y sigue estos pasos:

1. SI TIENE ZIP: Descomprime el archivo

2. SI NO TIENE NODE.JS:
   - Va a https://nodejs.org/
   - Descarga la versión LTS
   - Instala (siguiendo los pasos por defecto)
   - Reinicia la PC

3. EN LA CARPETA: Hace doble clic en "INSTALAR.bat"
   (o en "iniciar.bat" si lo anterior no funciona)

4. ¡LISTO! Se abre automáticamente


PASO 4: Soporte (opcional)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Ofrece:
  ✓ Primeros 30 días de soporte técnico
  ✓ Respuesta por WhatsApp/email
  ✓ Solución de problemas a distancia


================================================================================
                              ARCHIVOS CLAVE
================================================================================

Entrega TODOS estos archivos en la carpeta:

✓ INSTALAR.bat                    ← Cliente hace doble clic aquí
✓ GUIA_INSTALACION_CLIENTE.md     ← Instrucciones para el cliente
✓ iniciar.bat                     ← Alternativa si lo anterior falla
✓ package.json                    ← Dependencias
✓ server.js                       ← Servidor (NO modificar)
✓ public/                         ← Carpeta con HTML, CSS, JS
  ├── index.html
  ├── style.css
  └── script.js

NO incluyas:
✗ node_modules/ (se descargará automáticamente)
✗ agenda.db (se creará automáticamente)
✗ .git/ (a menos que uses GitHub)
